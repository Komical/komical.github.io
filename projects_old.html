A Rats Tale
Creators: JB and I
January 2025 - Present 		Our first official digital product. Built using the Godot Engine and Trenchbroom with a focus on level design and gameplay flow.

It is a puzzle game that was inspired by portal, we wanted to create a "valve" feel, we have always loved their games.

To create the levels for this game we used a program derived from the quake days called TrenchBroom. TrenchBroom uses a brush based model editor that makes it super easy to test, build, and test again in a short amount of time.
Once we got our pieces into TrenchBroom, it felt like playing with building blocks to create levels!
Chess the Card Game
Creators: JB and I
July 2025 - September 2025 		Our first dive into game development. This project helped us learn iteration, design fundamentals, and community engagement.

This game was an interesting mix between Magic The Gathering and Chess. It did much better than we thought it would!

As a studio, we want to create more card games down the line, hopefully in the digital space!
A lot of people bought table top simulator for this mod specifically, so one day we hope to put our own game on steam!
