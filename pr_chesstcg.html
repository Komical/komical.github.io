About The Project


Chess the Card Game is a hybrid strategy project combining traditional chess mechanics with collectible card game systems.


Players build decks, control board space, and use unit-based interactions to outmaneuver opponents.


The project was primarily developed and tested inside Tabletop Simulator with iterative balance updates.

Core Features


  • Deck-building strategy system
  • Chess-inspired movement rules
  • Competitive PvP gameplay
  • Board control mechanics
  • Tabletop Simulator prototyping
  • Community testing iteration loop
