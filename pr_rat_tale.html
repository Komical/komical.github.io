About The Project


A Rats Tale is a puzzle-focused first-person game inspired by classic Valve titles such as Portal and Half-Life.


The project was developed using the Godot Engine alongside TrenchBroom for rapid level iteration and brush-based map design.


Our main goal was to create satisfying environmental puzzles while maintaining a strong atmosphere and memorable level flow.

Core Features


  • First-person puzzle gameplay
  • Valve-inspired atmosphere
  • Custom level design pipeline
  • Brush-based environment creation
  • Fast gameplay iteration
  • Experimental mechanics
